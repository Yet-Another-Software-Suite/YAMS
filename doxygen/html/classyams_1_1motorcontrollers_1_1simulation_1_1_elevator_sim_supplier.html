Detailed Description

SimSupplier backed by a WPILib ElevatorSim.

Translates between the linear physics (metres, m/s) of the ElevatorSim and the angular mechanism representation (degrees, dps) expected by the SimSupplier interface. The circumference of the drive sprocket/drum is required to perform this conversion.

The duty cycle is read from the motor controller each iteration unless an explicit input voltage has been set.
