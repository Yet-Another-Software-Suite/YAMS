Detailed Description

Abstract base class for all YAMS motor controller wrappers.

Provides a hardware-agnostic interface for open-loop control, closed-loop setpoints, encoder reads/writes, simulation, telemetry, and SysId characterization. Concrete implementations wrap CTRE TalonFX/TalonFXS and REV SPARK Max/Flex.
