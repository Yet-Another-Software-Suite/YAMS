Detailed Description

Builder for a simulated yams::motorcontrollers::simulation::Sensor.

Mirrors the Java SensorConfig class. Add typed fields with WithField(), inject simulated values by match-time window or arbitrary trigger with WithSimulatedValue(), then call GetSensor() to obtain the ready-to-use Sensor instance (lazily constructed on first call).
