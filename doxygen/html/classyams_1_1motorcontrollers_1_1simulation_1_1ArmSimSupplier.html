Detailed Description

SimSupplier backed by a WPILib SingleJointedArmSim.

Suitable for arm mechanisms (Arm, DoubleJointedArm) that need gravity compensation in simulation. The duty cycle is read from the motor controller each iteration unless an explicit input voltage has been set.
