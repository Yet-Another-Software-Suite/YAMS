Detailed Description

Configuration for an LQR (Linear Quadratic Regulator) closed-loop controller.

Uses a fluent builder pattern; all With* methods return *this for chaining. After configuring the plant type, gains, and noise parameters call GetLoop() to obtain the ready-to-use LinearSystemLoop.
