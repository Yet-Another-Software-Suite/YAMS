Detailed Description

Configuration for an absolute encoder used to seed or synchronize a relative encoder.

Supports CTRE CANcoder, CANdi, REV SPARK absolute encoder, Through-bore encoders, and a CUSTOM type backed by an arbitrary angle supplier. Uses a fluent builder pattern; all With* methods return *this for chaining.
